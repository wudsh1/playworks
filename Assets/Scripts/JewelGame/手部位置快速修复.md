# 手部位置 - 快速修复

## ⚡ 1分钟解决方案

### 方法 1：开启世界坐标模式（推荐）✨

找到 `HandGuide` 对象 → Inspector → `JewelHandGuide` 组件：

```
【调试工具】
✅ Use World Position: true  ← 勾选这个！
✅ Show Debug Info: true     ← 开启调试
```

**立即运行游戏测试！**

---

## 🎯 方法 2：查看 Scene 视图调试

1. **打开 Scene 视图**（与 Game 视图并排显示）

2. **运行游戏**

3. **观察 Gizmos**：
   - 🟡 **黄色球** = 手部当前位置
   - 🟢 **绿色框** = 目标方块位置
   - 🔴 **红线** = 偏差

4. **如果红线很长**：
   - 方法A：开启 `Use World Position = true`
   - 方法B：调整 `X/Y Offset`

---

## 🔧 方法 3：手动微调

如果手部位置接近但不准确：

```
X Offset: 
  手部偏左 → +50
  手部偏右 → -50

Y Offset:
  手部偏下 → +50
  手部偏上 → -50
```

**建议**：每次调整 10-50 个单位

---

## 📱 方法 4：检查 Canvas 配置

选中 `Canvas` → Inspector → `Canvas Scaler`：

```
✅ UI Scale Mode: Scale With Screen Size
✅ Reference Resolution: 1080 x 1920（或其他合理值）
✅ Screen Match Mode: Match Width Or Height
✅ Match: 0.5
```

---

## 🧪 测试步骤

1. **运行游戏**
2. **在 HandGuide 组件上右键** → `打印调试信息`
3. **查看控制台输出**
4. **检查 Scene 视图的 Gizmos**

---

## 📊 预期结果

### ✅ 位置正确的标志

- Scene 视图中黄色球在绿色框内
- 红线很短或不可见
- Game 视图中手部在目标方块上
- 控制台输出的位置值合理

### ❌ 位置错误的标志

- Scene 视图中黄色球远离绿色框
- 红线很长
- Game 视图中手部偏离目标
- 控制台输出位置值异常（如超大数字）

---

## 💡 最快解决方案

### 90% 的问题可以这样解决：

**步骤 1**：
```
Use World Position: true
```

**步骤 2**：运行游戏测试

**步骤 3**：如果还有偏差，微调 `X/Y Offset`

**完成！** ✨
