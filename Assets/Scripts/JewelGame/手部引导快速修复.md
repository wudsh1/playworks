# 手部引导 - 快速修复指南

## ⚡ 5分钟快速修复

### 问题
❌ 手部引导在不同分辨率手机上位置不对

### 解决方案
✅ 已自动修复！只需确认配置

---

## 🚀 快速步骤

### Step 1: 确认自动调整已开启

找到 `HandGuide` 对象 → Inspector → `JewelHandGuide` 组件：

```
【高级设置】
✅ Auto Adjust Parent: true  ← 确保勾选
```

### Step 2: 运行游戏测试

1. **点击 Play 按钮**
2. **查看控制台日志**：
   ```
   [JewelHandGuide] 自动调整父物体到: Canvas
   [JewelHandGuide] 初始化完成
   ```
3. **观察手部引导位置是否正确**

### Step 3: 微调位置（如果需要）

如果位置还有轻微偏差：

#### 向右移动
```
X Offset: +30
```

#### 向左移动
```
X Offset: -30
```

#### 向上移动
```
Y Offset: +30
```

#### 向下移动
```
Y Offset: -30
```

**建议**：每次调整 10-50 个单位

3. **观察手部引导位置是否正确**

在不同设备上测试，确保位置正确

---

## 🐛 还是不对？

### 检查清单

- [ ] `Auto Adjust Parent` 已勾选
- [ ] 控制台没有错误日志
- [ ] `HandImage` 有正确的 Sprite 图片
- [ ] Canvas Scaler 设置正确（Scale With Screen Size）

### 查看详细日志

控制台应该输出：

```
[JewelHandGuide] 自动调整父物体到: Canvas
[JewelHandGuide] 初始化完成。游戏板容器: BoardContainer, 手部父物体: Canvas
[JewelHandGuide] 更新位置: (120.5, -345.2)
```

如果没有这些日志，检查：
- `JewelGameManager` 是否正确引用了 `HandGuide`
- `HandGuide.Initialize()` 是否被调用

## 🎯 推荐配置

```
【引导设置】
Guide Row: 3              // 第4行
Guide Start Column: 6     // 第7列
Guide End Column: 7       // 第8列
Move Duration: 1.0        // 移动速度

【高级设置】
Auto Adjust Parent: ✅ true
Y Offset: 0               // 根据需要微调
X Offset: 0               // 根据需要微调
```
